# Sales Dashboard in Power BI

## Objective
Create an interactive dashboard to monitor sales and profit performance across categories, sub-categories, and regions.

## Tools
- Power BI Desktop

## Dashboard Features
- KPI cards for total sales and total profit
- Profit analysis by category and sub-category
- Regional performance comparison
- Sales vs Profit scatter plot to identify inefficient sub-categories
- Interactive slicers for dynamic filtering

## Key Insights
- Technology is the most profitable category
- Some sub-categories show high sales but low profit
- Regional performance varies significantly

## Business Value
This dashboard helps business users quickly identify profit drivers and areas for optimization.
